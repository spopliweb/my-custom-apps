Frequently Asked Questions

Can I enter any currency amount?

Yes! The module reads the currency configured in your Odoo POS session. The quantity is calculated using the product's unit price and the amount you enter — works with any configured currency.

Does it work for non-weight products?

Absolutely. Any product in your POS catalogue works. The module divides the entered amount by unit price to compute the quantity, regardless of unit of measure.

Multiple POS sessions simultaneously?

Yes. Each POS session is isolated so multiple cashiers can use the feature simultaneously without any conflict.
