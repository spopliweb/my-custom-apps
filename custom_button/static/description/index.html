Frequently Asked Questions

Does this module work with Odoo Community Edition?
Yes! This module is fully compatible with both Odoo 18 Community and Enterprise editions. It requires only the standard point_of_sale module with no additional dependencies.
How does the quantity calculation work?
Simply type ₹500 worth of item, the system reads current POS price, calculates 500 ÷ Price/kg = Qty, then fills in the exact decimal quantity automatically. All calculations happen in real-time with zero delay.
Does it support multiple currencies?
Yes! The module automatically detects your Odoo system's currency settings and adapts accordingly. It supports 190+ currencies with proper decimal precision and regional formatting rules.
Will this affect my existing POS data?
No. This module only adds a new input method to your POS interface. Your existing products, orders, customers, and all data remain completely safe and untouched.
Can multiple cashiers use it simultaneously?
Yes! The module fully supports multiple concurrent POS sessions. Each cashier can use it independently on their own POS session without any conflicts.
