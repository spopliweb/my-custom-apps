POS Manual Amount Entry

This module allows cashiers to enter any purchase amount and automatically calculates the exact product quantity. Specially built for weight-based, bulk, and flexible pricing retail environments.

Key Features:

  • Smart Amount-Based Entry: Enter purchase amounts in any currency and let the system automatically calculate the exact quantity needed. Perfect for weight-based products like fruits, vegetables, spices, and bulk items.
  • Weight & Bulk Product Support: Specially designed for businesses selling by weight (kg/gram), volume (liters/ml), or measurement. Enter the customer's budget and get exact quantity instantly.
  • Multi-Currency Support: Works seamlessly with any currency configured in your Odoo system — USD, EUR, INR, GBP and 100+ more. Auto-detects system currency with proper decimal precision.
  • Touch-Optimized Interface: Large touch-friendly buttons designed for tablets, POS terminals, and mobile devices. Responsive layout adapts to any screen size.
  • Real-Time Processing: Lightning-fast calculations and instant order line updates with zero latency. Comprehensive error handling prevents any invalid input from causing issues.
  • Robust Validation & Safety: Advanced input validation and data integrity protection with user-friendly error messages. Fail-safe error handling built in.
  • One-Click Installation: Simple module installation through Odoo Apps Store or manual upload. Automatic dependency resolution with no server restart required.

Frequently Asked Questions

Does this module work on Odoo Community Edition?

Yes! This module works perfectly on both Odoo 18 Community and Enterprise editions. It is specifically tested on Odoo 18.0 and is backward compatible with Odoo 17.0. It does NOT work on Odoo Online due to platform restrictions on custom JavaScript modules.

How does the automatic quantity calculation work?

When you click "Enter Amount" and input a value (e.g., ₹250), the module divides that amount by the product's unit price to calculate the exact quantity. For example, if Bagel costs ₹1.25/kg and you enter ₹250, the system automatically sets the quantity to 200kg. The calculation happens instantly in real-time with high decimal precision.

Which currencies are supported?

The module supports all 100+ currencies configured in your Odoo system. It auto-detects the system currency and applies proper regional formatting and decimal precision rules. No additional configuration needed — it simply works with whatever currency your POS session is set to.

Will this module affect my existing POS data?

No — this module only adds a new button and popup to your POS interface. It does not modify any existing data, database tables, or POS workflows. Your existing orders, products, and configurations remain completely untouched. You can safely install and uninstall without any data risk.

Does it work with multiple POS sessions and users?

Yes! The module fully supports multiple concurrent POS sessions and users. Each session is fully isolated — calculations in one session do not affect another. User permission controls from standard Odoo POS apply. Real-time synchronization is maintained across all sessions through Odoo's standard architecture.
