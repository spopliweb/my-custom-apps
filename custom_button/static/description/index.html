Does this module work with Odoo Community Edition?

Yes! Fully compatible with Odoo 18 Community and Enterprise editions. Requires only the point_of_sale module — no extra dependencies needed.

How does the quantity calculation work?

Type ₹500 → system reads POS price → calculates 500 ÷ Price/kg = Qty → fills exact decimal quantity. All real-time, zero delay.

Does it support multiple currencies?

Yes! Auto-detects your Odoo system currency. Supports 190+ currencies with decimal precision and regional formatting rules.

Will this affect my existing POS data?

No. This module only adds a new input method. All existing products, orders, and customer data remain completely safe and untouched.

Can multiple cashiers use it simultaneously?

Yes! Multiple concurrent POS sessions are fully supported. Each cashier works independently without any conflicts.
