Does this work with Odoo Community Edition?

Yes! Fully compatible with Odoo 18 Community and Enterprise. Requires only point_of_sale — no extra dependencies.

How does the quantity calculation work?

Type ₹500 → system reads POS price → 500 ÷ Price/kg = Qty → fills exact decimal quantity. All real-time, zero delay.

Does it support multiple currencies?

Yes! Auto-detects your Odoo system currency. Supports 190+ currencies with decimal precision and regional formatting.

Will this affect my existing POS data?

No. Only adds a new input method. All existing products, orders, and customer data remain completely safe.

Can multiple cashiers use it simultaneously?

Yes! Multiple concurrent POS sessions fully supported. Each cashier works independently without conflicts.
